# Student_Management_System
A simple Command Line Interface (CLI) Student Management System built with Python. This project demonstrates how to manage student records by adding, searching, updating, deleting, and displaying student information while practicing core Python concepts.

**Features**

➕ Add student records

📋 Display all students

🔍 Search students by roll number

✏️ Update student roll numbers

❌ Delete student records

🔄 Generate student records using a generator

**Functions**

1-*args

2-**kwargs

3-Generators (yield)

4-Recursion

5-Lists

6-Dictionaries

7-Loops

8-Conditional Statements

**Learning Objective**

This project was developed to strengthen my understanding of Python fundamentals by implementing a real-world Student Management System. It demonstrates the practical use of functions, recursion, generators, *args, **kwargs, lists, dictionaries, and CRUD (Create, Read, Update, Delete) operations.
